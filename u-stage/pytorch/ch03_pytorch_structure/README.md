## PyTorch Project Template Overview
- 언제까지 jupyter notebook과 함께할래?
- 배포 및 공유 단계에서는 notebook 공유의 어려움
  - 쉬운 재현의 어려움, 실행 순서 꼬임
- DL 코드도 하나의 프로그램
  - 개발 용이성 확보와 유지보수 항상 필요

- 코드도 레고블럭처럼!
  - OOP + Module = Project!

- 다양한 프로젝트 템플릿이 존재함
  - 사용자 필요에 따라 수정하여 사용
